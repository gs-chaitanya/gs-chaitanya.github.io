I'm a final-year ECE undergrad at IIT (BHU) Varanasi, currently a visiting researcher at NUS working on FPGA-based hardware security. My research spans microarchitectural side-channel analysis, neuromorphic computing with ReRAM & FeFET, and open-source EDA tooling. Two-time Google Summer of Code contributor (OpenROAD & Apertium) and Linux Foundation Mentee.

Experience

  • Working towards developing FPGA-friendly approaches for microarchitectural side-channel analysis that eliminate the need for explicit Dynamic Information Flow Tracking.
  • Engaged in hardware security research to mitigate microarchitectural side-channel attacks by adapting the MileSan dynamic analysis framework for FPGA-accelerated RTL simulations.
  • Contributed to OpenROAD, an open-source RTL2GDSII flow for digital IC design. Implemented end-to-end support for rectilinear floorplans inside the flow.
  • Modified user-facing TCL scripts, intermediate SWIG wrappers, and the backend C++ logic to support polygonal floorplans.
  • Designed and implemented a secure two-stage bootloader architecture with Root of Trust and Digital Signature verification.
  • Incorporated AES decryption capabilities to protect sensitive IP in compiled firmware binaries.
  • Optimized the bootloader for low runtime memory usage and heap utilization for resource-constrained environments.
  • Developed a Hardware Abstraction Layer Transitional Library for Software to Hardware Posits for RISC-V. Selected for the Spring 2025 term of the Linux Foundation Mentorship Program.
  • Improved and delivered a new version of the SoftPosit-cpp Library with native posit arithmetic; implemented math functions for 8, 16, and 32-bit posits with benchmarks and unit tests.
  • Created CI/CD pipelines for automated building and packaging, a Python Wrapper for SoftPosits, and a Debian Package.
  • Worked on creating an ABC script to improve synthesis options in Yosys for creating a balanced synthesis plan, balancing between Speed and Area optimized synthesis.
  • Contributed to Apertium, an open-source machine translation platform. Developed core tools with CLI utilities for automated bidirectional dictionary generation from parallel translated texts.
  • Experimented with word alignment methods including Bayesian models with Markov Chain Monte Carlo and Graph Neural Networks for multiparallel word alignment.
  • Developed Verilog and Embedded C applications using Vivado and Xilinx SDK on a Digilent Zybo FPGA board for an FPGA-based controller to perform Read, Write, and Erase operations on 2D and 3D SLC and MLC raw NAND Flash Chips.
  • Used bit-banging to precisely control onboard GPIO pins via the ONFI Consortium Protocol. Developed programs in Embedded C on Embedded Linux on an Altera DE1 SoC.
