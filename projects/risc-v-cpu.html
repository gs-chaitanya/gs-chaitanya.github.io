Overview

Designed and implemented a 32-bit RISC-V CPU supporting the RV32IM instruction set (base integer + hardware multiply/divide) in Verilog HDL. The design features a 5-stage pipeline (IF/ID/EX/MEM/WB) with hazard detection and forwarding logic, validated via extensive RTL testbenches and cross-compiled C benchmarks.

Key Challenges

  • Correctly handling data and control hazards in the pipeline — forwarding paths were verified against all RAW/WAW dependency combinations.
  • Multiply/divide in the M extension required multi-cycle execution; the pipeline stall logic was carefully designed to avoid liveness violations.
  • Timing closure at 130nm required pipeline restructuring to meet setup constraints on the critical path through the ALU.
